Aplicación orientada a la realización de ejercicios de fisioterapia o rehabilitación mediante una webcam.
La persona responsable del paciente puede crear ejercicios personalizados, sesiones de ejercicios y consultar los datos de sus pacientes.
El paciente puede realizar estas sesiones de ejercicios o consultar sus propios resultados.
Se ha desarrollado en Windows 10 mediante la versión de Python 3.10.11.

Para inicializar y descargar todas las dependencias necesarias (se pueden consultar también en el archivo requeriments.txt) se pone a su disposión tanto un archivo Instalacion.bat como uno Instalacion.sh, dependiendo de si se está empleando Windows o Linux respectivamente. Si ya tiene todo lo necesario descargado, puede ejecutar el fichero GUI.py directamente.
